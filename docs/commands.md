* ```() = något värde (true|false|sträng...)```
* ```[] = friviligt```

    <dd>check_ram kollar RAM användningen på klienten.
        <table>
            <tr>
                <th>Flagga</th>
                <th>Värde</th>
                <th>Beskrivning</th>
            </tr>
            <tr>
                <td>-total</td>
                <td>Ingenting</td>

            </tr>
         </table>
    </dd>
</dl>
